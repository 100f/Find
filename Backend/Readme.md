# Guia para ativação do código

- Atualizar as informações de usuário e senha no arquivo "knexfile.js"

- Acessar pasta "Backend"
```node
cd Backend
```

- Instalar pacotes do node modules(caso não tenha) 
```node
npm i
```

- Inicializar o servidor
```node
npm run dev
```

- Criar tabelas
```node
npm run migrations
```

- Popular tabelas 
```node
npm run seeds
```
